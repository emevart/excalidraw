# Changelog

Все заметные изменения в SdamExDraw документируются здесь.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/). Версионирование: [Semantic Versioning](https://semver.org/).

## [0.26.64] - 2026-03-19

### Исправлено

- Сглаживание кривых: уменьшен радиус smoothing 8→3 для минимизации сжатия дуг

## [0.26.63] - 2026-03-19

### Исправлено

- Спрямление прямых линий: сохраняет все точки (проекция) вместо 2 финальных — рендерер не сужает штрих

## [0.26.62] - 2026-03-19

### Исправлено

- Спрямление кривых: moving average сглаживание вместо RDP simplification — сохраняет форму и длину кривой
- Спрямление прямых: проекция на линию start→end

## [0.26.61] - 2026-03-19

### Исправлено

- Спрямление: линия больше не сжимается — проекция на прямую вместо arc length parameterization

## [0.26.60] - 2026-03-19

### Исправлено

- Спрямление: сброс состояния после анимации (повторные штрихи не блокируются)
- Спрямление: конец линии остаётся под курсором (не добавляется лишняя точка при pointerUp)
- Миникарта: кнопка переключения слева от зума в одном ряду
- Миникарта: точный рендеринг фигур вместо прямоугольников (freedraw, линии, эллипсы, ромбы)

## [0.26.59] - 2026-03-19

### Добавлено

- Спрямление линии (hold-to-straighten): удержание карандаша без движения 500мс спрямляет штрих (прямая или RDP-сглаженная кривая) с анимацией
- Drag & drop картинки из карточки задания на доску (`text/uri-list` URL → fetch → insert)
- Миникарта навигации: переключаемая мини-карта рядом с зумом, клик/drag для перемещения по канвасу

## [0.26.58] - 2026-03-19

### Добавлено

- `history.undo()` и `history.redo()` в ExcalidrawImperativeAPI для программного вызова из внешних тулбаров

## [0.26.57] - 2026-03-19

### Добавлено

- Двухуровневый мобильный тулбар: компактный нижний ряд (категории) + верхний ряд настроек с вариантами инструментов
- Все варианты фигур, линий, карандаша/маркера, выделения доступны в settings row с горизонтальным скроллом
- Фигуры отсортированы по категориям: базовые → четырёхугольники → многоугольники | призмы → пирамиды → тела вращения
- Тонкий скроллбар на десктопе при узких экранах (`@media (pointer: fine)`)

### Изменено

- Убраны ToolPopover дропдауны для фигур, линий, карандаша — заменены плоскими кнопками в settings row
- Settings row: цвета + кнопка свойств (⚙) + варианты инструментов inline

### Исправлено

- Позиционирование поповеров привязано к тулбар-контейнеру через Popover.Anchor virtualRef
- Перекрытие дропдаунов с панелью shape actions на мобилке
- PropertiesPopover открывается вверх на мобилке (не перекрывает тулбар)
- Z-index ToolPopover и extra tools dropdown поднят до ui-styles-popup

## [0.26.55] - 2026-03-19

### Исправлено

- Highlighter: диапазон слайдера 8-40 с шагом 2, strokeWidth по умолчанию 12

## [0.26.54] - 2026-03-19

### Добавлено

- Highlighter: режим freedraw с переключателем карандаш/маркер
- Три независимых набора настроек: pencil, highlighter, shape
- Слайдер толщины линии с превью вместо 3 radio-кнопок
- Поддержка хайлайтера в мобильном тулбаре

## [0.26.53] - 2026-03-18

### Добавлено

- Кастомные тултипы вместо нативного `title=` (400ms задержка, 11px шрифт)
- Hover-эффекты на всех кнопках
- Поддержка Apple Pencil hover

## [0.26.52] - 2026-03-17

### Добавлено

- 14 пресетов фигур в мобильном тулбаре (7 2D + 7 3D)
- Линия как инструмент по умолчанию для linear tools
- Two-finger double-tap для undo на тачскрине

### Исправлено

- Bounding box для 2D polygon пресетов на мобилке
- Transform handles для polygon пресетов

## [0.26.51] - 2026-03-16

### Добавлено

- Wireframe (3D preset) UX: перетаскивание вершин, курсор move на вершинах
- Расширенная зона захвата рёбер (10px)
- Блокировка входа в группу по двойному клику для wireframe
- Перетаскиваемая вершина конуса (shared vertex ID "APEX")

## [0.26.50] - 2026-03-15

### Добавлено

- Компактная панель стилей для всех устройств кроме телефонов
- Русская раскладка ЙЦУКЕН — горячие клавиши работают
- Настройки в гамбургер-меню: grid toggle, grid snap toggle
- Защита от краша в `_renderInteractiveScene`
- Полная русская локализация (все ключи + 13 исправлений)
- Выравнивание zoom controls через `--editor-container-padding`

### Исправлено

- Безопасный линейный редактор (проверка selectedLinearElement)
- ConfirmDialog не занимает весь экран в компактном режиме
- Рёбра треугольной призмы: правое боковое и верхнее левое — сплошные
- TS 5.7: ArrayBuffer/BufferSource type assertions
