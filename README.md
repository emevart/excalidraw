# SdamExDraw

Интерактивная доска для решения задач на [sdamex.com](https://sdamex.com).

React-компонент, встраиваемый в платформу SdamEx. Заточен под математику и геометрию, оптимизирован для мобильных устройств и стилуса.

Публикуется как `@emevart/excalidraw` в GitHub Packages.

## Возможности

**Геометрия и 3D-фигуры**

- 14 пресетов фигур (7 плоских + 7 объемных): призмы, конусы, пирамиды, цилиндры
- Wireframe-режим с перетаскиванием вершин и общими точками
- Перетаскиваемая вершина конуса

**Мобильный UX**

- Адаптивный тулбар со всеми пресетами фигур
- Two-finger double-tap для undo
- Компактная панель свойств для всех устройств
- Позиционирование дропдаунов и попаверов без перекрытий

**Инструменты рисования**

- Хайлайтер с независимыми настройками (цвет, толщина, прозрачность)
- Слайдер толщины линии с превью
- Улучшенный рендеринг freedraw через laser-pointer (без артефактов на углах)

**Локализация и ввод**

- Полная русская локализация (все ключи + 13 исправлений качества)
- Горячие клавиши работают на раскладке ЙЦУКЕН

**Стабильность**

- Защита от краша при рендере (try-catch в interactiveScene)
- Безопасный линейный редактор
- Фиксы типов для TypeScript 5.7

## Установка

```bash
# .npmrc
@emevart:registry=https://npm.pkg.github.com

# установка
npm install @emevart/excalidraw
```

## Разработка

```bash
yarn fix              # линтинг + форматирование (0 warnings)
yarn test:typecheck   # проверка типов
cd packages/excalidraw && yarn build:esm   # сборка пакета
```

## Структура

- `packages/excalidraw/` — основная библиотека, публикуется как `@emevart/excalidraw`
- `packages/common/`, `packages/element/`, `packages/math/`, `packages/utils/` — внутренние пакеты
- `excalidraw-app/` — демо-приложение (не используется)

## На основе

Создано на основе [Excalidraw](https://github.com/excalidraw/excalidraw) (MIT).
